Passion project code structure for the best possible robot for the FRC 2026 Rebuilt season

Goals:
- Use a dye-rotor for indexer
- use a cubic form factor
- has a linear intake with a "slap-down" roller
- will only use kraken motors
- use a turret
- use MK5 swerve modules
- design will also be published, created in OnShape
- Will have a an LED system
- as full of an FOV as possible
- Will use Ardu cams and Orange Pi 5's for vision
- 2 VH radios using one as 6 GHz and the other being 2.4 GHz
